# AI Экспедиция 🎓

Интерактивное обучение-игра по всем возможностям Claude: от «ни разу не открывал» до практикующего специалиста (Claude Code, скиллы, плагины, MCP, API).

Всё обучение построено на **официальных документах Anthropic** — каждый урок привязан к первоисточнику.

## Документы проекта

| Файл | Что это |
|---|---|
| [docs/PLAN.md](docs/PLAN.md) | План проекта: концепция, варианты форматов, архитектура, дорожная карта |
| [docs/CURRICULUM.md](docs/CURRICULUM.md) | Учебная программа: 8 миров, 72 урока, треки «с нуля» и «специалист» |
| [docs/SOURCES.md](docs/SOURCES.md) | Реестр официальных первоисточников (документация, библиотеки, курсы) |

## Концепция коротко

- **Карта миров** в стиле Duolingo: 8 миров от основ Claude до API/Agent SDK
- **6 типов заданий:** квиз, «собери промпт», «найди ошибку», симулятор чата, симулятор терминала Claude Code, реальные миссии в настоящем Claude
- **Геймификация:** XP, уровни, стрики, бейджи, альбом карточек-функций, боссы миров
- **Два входа:** трек «с нуля» и экспресс-трек для практиков (испытания миров + входной тест)
- **Стек:** React 18 + TypeScript + Vite + Tailwind, контент как JSON-данные, MVP без бэкенда

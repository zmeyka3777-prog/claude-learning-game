# Юридические документы проекта

Рабочие **черновики** для продажи доступа. Это шаблоны, а не юридическое заключение —
перед публичным запуском проверьте у юриста две точки: формулировку по товарному знаку
и обязательства по персональным данным (152-ФЗ / GDPR).

| Файл | Назначение |
|---|---|
| [OFERTA.md](./OFERTA.md) | Публичная оферта (договор с покупателем: предмет, цена, возврат, ответственность) |
| [POLITIKA_KONFIDENTSIALNOSTI.md](./POLITIKA_KONFIDENTSIALNOSTI.md) | Политика конфиденциальности (какие данные, цели, права пользователя) |
| [PRAVOVAYA_INFORMATSIYA.md](./PRAVOVAYA_INFORMATSIYA.md) | Дисклеймер о неаффилированности, товарные знаки, порядок takedown |

## Что сделать перед запуском продажи

1. Заменить все поля `[…]` своими данными (ФИО, статус самозанятого, ИНН, цена, email, даты).
2. Оформить статус самозанятого (НПД) до первого платежа.
3. Разместить документы по публичным ссылкам и сослаться на них на странице оплаты
   (галочка «согласен с офертой и политикой» — непреднажатая).
4. Полный разбор рисков — в [../docs/LEGAL_REVIEW.md](../docs/LEGAL_REVIEW.md).

Дисклеймер о неаффилированности уже выведен в подвале игры на обоих языках.

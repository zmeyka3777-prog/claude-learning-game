## 2. Какие данные обрабатываются

**2.1. Данные, хранящиеся только на устройстве пользователя (не передаются Оператору).**
Игровой прогресс (пройденные уроки, XP, стрик, бейджи, выбранный язык и трек, имя для сертификата)
сохраняется **локально в браузере (localStorage)**. Эти данные **не покидают устройство**, не
передаются на серверы Оператора и не собираются им. Пользователь может удалить их, очистив данные сайта
в браузере.

**2.2. Аналитика использования.**
Проект использует систему веб-аналитики **[PostHog]** для сбора **обезличенной** статистики
использования (какие экраны открываются, какие уроки проходятся, где возникают трудности). При этом
может обрабатываться техническая информация: IP-адрес, тип устройства и браузера, страна.
Правовое основание — **согласие пользователя** (для ЕС — предварительное согласие). Цель — улучшение
качества обучения. Аналитика **не используется** для идентификации личности и не связывается с именем.

**2.3. Данные при оплате.**
Оплата доступа (если применимо) проводится через сторонний платёжный сервис **[Продамус / Lava.top]**.
Платёжные данные (карта и т.п.) вводятся **на стороне платёжного сервиса**, Оператор их **не получает и
не хранит**. Оператор может получить от сервиса факт оплаты и контакт покупателя (email) для
предоставления доступа и поддержки. Обработка платёжным сервисом регулируется его собственной политикой.

хостинг), исключительно в рамках указанных целей. Оператор не продаёт персональные данные.

**Трансграничная передача.** Часть сервисов (например, аналитика **[PostHog]**) может размещаться за
пределами РФ. При наличии такой передачи она осуществляется с согласия пользователя. *(При добавлении

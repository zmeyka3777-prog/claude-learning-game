# Дизайн-система «Neural Galaxy»

Тема создана на базе пресета Midnight Galaxy (theme-factory) и развита под игровой продукт.
Настроение: глубокий космос, неоновое свечение, стекло, ощущение экспедиции и прогресса.

## 1. Палитра

### Фоны
| Токен | Значение | Использование |
|---|---|---|
| `--bg-deep` | `#0B0E1A` | основной фон приложения |
| `--bg-nebula` | `#131A33` | вторая ступень градиента фона |
| `--bg-card` | `rgba(255,255,255,0.06)` | стеклянные карточки |
| `--bg-card-hover` | `rgba(255,255,255,0.10)` | hover карточек |
| `--border-glass` | `rgba(255,255,255,0.12)` | границы стекла |

Фон приложения: `radial-gradient(ellipse at top, #1B2350 0%, #131A33 40%, #0B0E1A 100%)` + слой звёзд (CSS-частицы, 2 слоя параллакса, медленное мерцание).

### Акценты
| Токен | Значение | Использование |
|---|---|---|
| `--accent-violet` | `#8B5CF6` | главный акцент, кнопки, активные узлы |
| `--accent-cyan` | `#22D3EE` | вторичный акцент, ссылки, прогресс |
| `--accent-amber` | `#F59E0B` | XP, награды, звёзды |
| `--accent-pink` | `#EC4899` | стрики, огонь |
| `--success` | `#34D399` | верные ответы, завершённые уроки |
| `--error` | `#F87171` | ошибки |

Фирменный градиент: `linear-gradient(135deg, #8B5CF6 0%, #6366F1 50%, #22D3EE 100%)` — заголовки-градиенты, кнопки CTA, прогресс-бары, рамки босс-уровней.

### Текст
- `--text-primary`: `#EEF1FF`
- `--text-secondary`: `#A8B0D3`
- `--text-muted`: `#6B7399`

Контраст: primary на bg-deep ≥ 12:1, secondary ≥ 5:1.

## 2. Типографика (обязательно с кириллицей)

| Роль | Шрифт | Fallback |
|---|---|---|
| Дисплей/заголовки | **Unbounded** (Google Fonts, cyrillic) | Manrope, system-ui |
| Текст | **Manrope** (cyrillic) | system-ui |
| Код/терминал | **JetBrains Mono** (cyrillic) | monospace |

Подключение через `@fontsource` пакеты (self-hosted, без CDN). Размеры: h1 32–40px (Unbounded 600), h2 24px, body 16px/1.7, код 14px.

## 3. Компоненты

### Стеклянная карточка (базовый примитив)
`background: var(--bg-card); backdrop-filter: blur(20px); border: 1px solid var(--border-glass); border-radius: 20px;`
Свечение при hover: `box-shadow: 0 0 40px rgba(139,92,246,0.25)`.

### Карта мира
- Вертикальная «звёздная тропа»: узлы-уроки соединены светящейся SVG-кривой.
- Узел урока: круг 64px, состояние: заблокирован (тусклый, замок) / доступен (пульсирующее свечение violet) / пройден (заполнен градиентом, галочка) / босс (72px, амбер-рамка, корона).
- Заголовок мира — стеклянная плашка с номером и градиентным заголовком.

### Экран урока
- Верх: прогресс-бар из сегментов (по числу шагов), градиентная заливка, анимация заполнения.
- Теория: блоки в стеклянных карточках; типы блоков — текст, код (тёмная панель с JetBrains Mono и кнопкой «копировать»), callout (💡 совет / ⚠️ важно / 📖 из документации — с рамкой цвета типа), таблица, «попробуй сам».
- Задания: крупные варианты-карточки; верный ответ — вспышка success + микро-конфетти; неверный — shake-анимация + объяснение.
- Внизу — кнопка «Читать оригинал» со ссылкой на первоисточник.

### Терминал-симулятор
Окно macOS-стиля (три точки), фон `#0D1117`, зелёный/белый текст JetBrains Mono, мигающий курсор, ввод с автофокусом.

### Чат-симулятор
Стилизация под claude.ai: пузыри пользователя справа (violet glass), ответы «Claude» слева, аватар-звёздочка, эффект печати (typewriter).

### Геймификация
- XP-тост: всплывает «+50 XP» с амбер-свечением и разлетающимися частицами.
- Стрик: пламя (pink→amber градиент), счётчик дней.
- Бейдж: шестиугольник со свечением, анимация выдачи — scale + rotate + конфетти.
- Карточка функции: «голографическая» карта с градиентной рамкой по редкости (обычная — серая, редкая — cyan, эпическая — violet, легендарная — amber) и лёгким 3D-tilt при наведении.
- Завершение урока: полноэкранный оверлей — звёздный взрыв, итог XP, награды, кнопка «Дальше».

## 4. Анимации (Motion)

- Появление экранов: fade + slide-up 24px, stagger детей 60мс.
- Узлы карты: пульс доступного узла (scale 1→1.06, 2с, ease-in-out, infinite).
- Прогресс-бары: spring-заполнение.
- Уважать `prefers-reduced-motion`: отключать пульсы/частицы.

## 5. Адаптивность

Mobile-first: 360px+. Карта — одна колонка; урок — max-width 720px по центру; на десктопе справа панель прогресса/наград (320px).

## 6. Тон текстов

Дружелюбный наставник на «ты», без канцелярита. Игрок = «исследователь», уроки = «экспедиции», миры = «сектора галактики». Но термины Claude — строго точные, как в документации.

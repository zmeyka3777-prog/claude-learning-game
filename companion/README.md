# Наставник Академии — скилл-компаньон для настоящего Claude

Это скилл, который превращает настоящий Claude (claude.ai или Claude Code) в наставника игры «AI Экспедиция». Установи его — и Claude будет:

- **проверять твои реальные миссии** из игры по чек-листу, честно, пункт за пунктом, с вердиктом «✅ МИССИЯ ЗАЧТЕНА» или «🔄 ДОРАБОТАЙ»;
- **давать задания сложнее** по любой теме игры — с точными ссылками на официальную документацию;
- **объяснять темы заново** — другими словами, с новым примером и мини-квизом из 3 вопросов.

В игре у каждой реальной миссии есть кнопка «Скопировать миссию для наставника» — она кладёт в буфер инструкцию миссии и чек-лист. Вставь это в чат с Claude, покажи свой результат — и наставник проверит.

## Установка

### Способ 1: claude.ai (браузер или приложение)

1. Открой claude.ai → **Настройки** (Settings) → раздел **Возможности / Skills**.
2. Создай свой скилл (Create skill / Add custom skill).
3. Скопируй содержимое файла [`SKILL.md`](./SKILL.md) целиком и вставь его в свой скилл. Название: `academy-mentor`.
4. Сохрани. Теперь в любом чате скажи «проверь миссию» — и Claude включит режим наставника.

### Способ 2: Claude Code (терминал / IDE)

Скопируй `SKILL.md` в папку скиллов своего проекта:

```bash
mkdir -p <проект>/.claude/skills/academy-mentor
cp SKILL.md <проект>/.claude/skills/academy-mentor/SKILL.md
```

Claude Code подхватит скилл автоматически при следующей сессии в этом проекте. Скилл сработает сам, когда ты напишешь «проверь миссию» или «наставник академии».

## Пример диалога

> **Ты:** Проверь миссию из AI Экспедиции по этому чек-листу.
>
> Миссия: создай свой первый проект. 1. Открой claude.ai и создай проект по реальной задаче… (вставлено кнопкой из игры)
>
> Проверь себя:
> - Я создал проект с осмысленным названием
> - Я написал инструкции: роль, контекст, стиль, формат
> - Я загрузил в базу знаний хотя бы один файл
>
> Вот скриншот моего проекта: [скриншот]

> **Claude (наставник):** Проверяю миссию «Первый проект». 🔭
>
> 1. Проект с осмысленным названием — ✅ вижу проект «Мой блог», отлично.
> 2. Инструкции: роль, контекст, стиль, формат — ❌ на скриншоте есть роль и контекст, но не задан стиль и формат ответов. Подсказка: добавь 1–2 предложения — каким тоном отвечать и в каком виде выдавать результат (список? markdown?).
> 3. Файл в базе знаний — ✅ загружен `план-постов.md`.
>
> Ты почти у цели — два пункта из трёх уже зачтены!
>
> 🔄 ДОРАБОТАЙ

После доработки пришли результат ещё раз — и получишь свой «✅ МИССИЯ ЗАЧТЕНА».
